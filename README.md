# 🎓 Student Management System

A console-based Java application developed to manage student records efficiently using SQL database connectivity. This project demonstrates CRUD operations, database handling, and object-oriented programming concepts in Java.

---

## 🚀 Features

- ➕ Add new student records
- ✏️ Update student details
- ❌ Delete student records
- 📋 Display all students
- 🔍 Search student information
- 📊 Sort student records
- 🗄️ SQL database integration

---

## 🛠️ Technologies Used

- Java
- JDBC
- SQL / MySQL
- Object-Oriented Programming (OOP)
